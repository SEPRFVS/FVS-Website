TaxE

A Game about Trains

Key Features - User

  • Over 4000 possible goals for hours of enjoyment
  • 10 different train types for increased competition
  • Visit over 30 european destinations
  • Freedom to move around; send your trains anywhere with easy routing
  • Competition to the end with blind goal completion

Key Features - Developer

  • Package and Class Hierarchy make it easy to find the code you need
  • Game data is stored in JSON format, so you can easily modify all data using a variety of tools (Map Editor included)
  • Compliant with all LibGDX standards so code follows through to on-screen
  • Gradle configures all dependencies automatically
  • JUnit Tests included to allow you to test out any modifications you make
