FAQs

  • How do I install for Android

    You'll need to enable "Applications from Unknown Sources" in your security settings before installing Trains across Europe. We don't distribute via Google Play so the only way to install is via our website.

  • What are the minimum requirements?

    The game will run on most machines capable of running Java 7, but we'd reccomend having 1GB of RAM for desktop. On Android, a screen size bigger than 5" will make the game more enjoyable.
