TaxE - A Game about Trains

"TaxE is the best game I've played in the past 20 minutes"
Developer

TaxE is the latest, and first, game from Team FVS. A story of great suspense, tragedy and jubilance as you must step up and become a train company. Your rivals will try to beat you and leave you with nowhere left to run but you must ensure your transportation empire prevails.

Download TaxE

TaxE runs on Windows, Mac and Linux with Java installed. It is not currently compatible with Android or iOS (and most probably never will be). Download below and Play Now.

Explore the Code

TaxE is open source, which means you're free to take the game resources and code and make your own version. All of our code is available to download on GitHub so take a look.
