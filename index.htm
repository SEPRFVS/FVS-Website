FAQs

  1. Oh no, not you again?

    Yes, it is. And we've done things! More precisely, we've adapted the game made by ADB in the previous assessment.

  2. What have you done to it?

    We've added some exciting new features as well as changing some old ones, such as

    • Adding obstacles to make gameplay more interesting at junctions and stations
    • Making scoring more challenging with the computer marking you against the best route with Dijkstra's algorithm
    • Adding more objectives to make the game more interesting, such as bonuses for getting to your destination on time or visiting another station
    • Making the game winnable by being the first to reach 6000 points
    • Removing clutter and making the game screen simpler with less non-gameplay functionality

    If you're a developer, you'll be pleased to hear that we've modified some of the code to give the program a solid foundation throughout. You can modify and extend the game easily to have even more features.

  3. What's available if I choose this?

    The same as last time, we'll provide you will all the documentation and files you need to build upon the game. We'll also provide you with an introduction to the game and our changes if you need it. We'll also be in email contact if you need us.

    We'd like to thank our mothers for making all of this possible, as well as our Hollywood publicist and all the people on the comittee who decided to give us this award. LibGDX (our Game Library), Eclise and JetBrains (our Development tools), Apache Commons (our Faster Fonts people) and GitHub (our Sourcecode Repository) should probably also get a passing mention.
